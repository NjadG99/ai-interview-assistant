1. Q: Explain JDK, JRE and JVM?
   A: JDK stands for Java Development Kit is a software development kit whereas JRE expand as Java Runtime Environment is a software bundle that allows Java program to run, whereas JVM(Java Virtual Machine) is an environment for executing byte code.

2. Q: Explain public static void main(String args[]) in Java?
   A: Public static void main (String[] args) Java main method is the entry point of any java program. Its syntax is always public static void main (String[] args) and can change the name of String array argument.

3. Q: Why Java is platform independent?
   A: Java is platform-independent because it does not depend on any type of platform. Hence, Java is platform-independent language and the programs are compiled into byte code and that byte code is platform-independent.

4. Q:  Why Java is not 100% Object-oriented?
   A: Java is not a pure Object oriented language, but so called a ‚ÄúHybrid‚Äù language. It supports primitive data type such as int ,char, float .All these are not object but you can argue we have wrapper class for each of the data type such Integer(int) , Float(float) etc . 

5. Q: What are wrapper classes in Java?
   A: A Wrapper class is a class whose object wraps or contains primitive data types. When we create an object to a wrapper class, it contains a field and in this field, we can store primitive data types.

6. Q: What are constructors in Java?
   A: Constructor is a block of codes similar to the method. It is called when an instance of the class is created. It is a special type of method which is used to initialize the object. 

7. Q: What is the difference between Array list and vector in Java?
   A: Vector and ArrayList both uses Array internally as data structure. They are dynamically resizable. But, ArrayList increases by half of its size when its size is increased.

8. Q: What is the difference between equals() and == in Java?
   A: equals() method for content comparison. == checks if both objects point to the same memory location whereas . equals() evaluates to the comparison of values in the objects.

9. Q: What are the differences between Heap and Stack Memory in Java?
   A: The major difference between Stack memory and heap memory is that the stack is used to store the order of method execution and local variables while the heap memory stores the objects and it uses dynamic memory allocation and deallocation.

10.Q: What is a package in Java? 
   A: Java package is used to categorize the classes and interfaces so that they can be easily maintained. It provides access protection and removes naming collision.
