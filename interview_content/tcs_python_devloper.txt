1. Q: Difference between Shallow Copy and Deep Copy?
   A: Shallow Copy: Creates a new compound object, but then inserts references into it to the objects found in the original.
      Deep Copy: Creates a new compound object and then recursively inserts copies of the objects found in the original. 

2. Q: Explain Inheritance in Python.
   A: Inheritance is an OOP concept where a new class (child/derived class) inherits attributes and methods from an existing class (parent/base class).

3. Q: What is the purpose of the __init__ method?
   A: The __init__ method is a special method in Python classes, known as the constructor. It is automatically called when a new instance of the class is created, allowing for initialization of the object's attributes.

4. Q: How do you handle exceptions in Python?
   A: Exceptions are handled using try, except, else, and finally blocks.
       try: Code that might raise an exception.
       except: Code to execute if a specific exception occurs.
       else: Code to execute if no exception occurs in the try block.
       finally: Code that always executes, regardless of whether an exception occurred.

5. Q: What is self in Python?
   A: self is a convention in Python to refer to the instance of a class within its methods. It is the first parameter in instance methods and allows access to the instance's attributes and methods.

6. Q: What are Generators in Python?
   A: Generators are functions that return an iterator. They use the yield keyword to produce a sequence of values one at a time, allowing for memory-efficient iteration over large datasets.

7. Q: How is memory managed in Python?
   A: Python uses a private heap for memory management. It employs reference counting and a garbage collector to reclaim memory from objects that are no longer referenced.

8. Q: What are Decorators?
   A: Decorators are functions that take another function as an argument, add some functionality, and return the modified function. They are used to extend or modify the behavior of functions or methods.

9. Q: Difference between List and Tuple:
   A: List: Mutable, ordered collection of items. Defined using square brackets [].
      Tuple: Immutable, ordered collection of items. Defined using parentheses ().

10.Q: Explain *args and **kwargs.
   A: *args: Used to pass a variable number of non-keyword arguments to a function as a tuple.
      **kwargs: Used to pass a variable number of keyword arguments to a function as a dictionary.
