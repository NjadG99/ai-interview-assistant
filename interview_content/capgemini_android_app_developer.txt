   A: Coroutines are lightweight threads in Kotlin that simplify asynchronous programming by allowing non-blocking code execution.

4. Q: Explain the different launch modes available in Android.
   A: Launch modes determine how a new instance of an activity is associated with the current task.
      -Standard: Creates a new instance of the activity in the task's stack.
      -SingleTop: If an instance of the activity already exists at the top of the stack, it will be reused.
