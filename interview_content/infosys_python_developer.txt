2. Q: How do you set a global variable in a function?
   A: Normally, when you create a variable inside a function, that variable is local, and can only be used inside that function. To create a global variable inside a function, you can use the global keyword.

3. Q: What are the rules for local and global variables in Python?
   A: In python, the variables referenced inside a function are global.
      When a variable is assigned new value anywhere in the body of a function then it is assumed as local.

   A: In Python, to write an empty class pass statement is used. Pass is a special statement in Python that does nothing. It only works as a dummy statement.
